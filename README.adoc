= AI Futures Guide
:toc: left
:toclevels: 2

AI Futures Guide is an open educational project dedicated to the study of
artificial intelligence, AI governance, AI safety, long-term scenarios,
and the societal impact of advanced AI systems.

The repository contains original educational materials, analytical books,
technical documentation, glossaries, diagrams and curated references.

The project is developed as an open knowledge base rather than a single
book.

== Goals

The primary goals are:

* explain complex AI concepts in accessible language;
* analyze major books and research reports;
* separate facts from forecasts and opinions;
* collect reliable references in one place;
* build an open educational resource that can evolve over time.

== Books

=== AI 2040

Status: in progress.

An analytical guide to the scenario document
"AI 2040: Plan A".

The book explains:

* the main ideas;
* historical context;
* technical background;
* AI governance;
* AI safety;
* alternative viewpoints.

Additional books will be added as the project develops.

== Principles

The project follows several principles.

=== Evidence first

Whenever possible, factual statements should be supported by reliable
sources.

=== Independent analysis

Books contained in this repository are independent analytical works.

They are not translations of third-party publications unless explicitly
stated.

=== Open knowledge

Educational material should remain openly available whenever possible.
